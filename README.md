# 🌍 GeoChallenge Daily

Adivina un país diferente cada día basándote en 4 pistas curiosas. De experto a fácil. Un proyecto de **Vibe Coding** nacido de la pasión por viajar y el deseo de descubrir el mundo.

🚀 **[Juega GeoChallenge aquí](https://geochallengedaily.lovable.app/)**

---

## 💡 Sobre el Proyecto y Motivación

Este proyecto nace de mi amor por viajar y mi curiosidad insaciable por descubrir datos únicos de cada rincón del planeta. **GeoChallenge** es una herramienta para promover la cultura general y enseñar datos fascinantes sobre países que quizás no conocías.

Ha sido desarrollado bajo la metodología de **Vibe Coding**, liderando un flujo de trabajo que combina la potencia de **Claude** para la ingeniería de prompts y **Lovable** para la ejecución y despliegue. Mi enfoque fue diseñar un reto diario que convierta el aprendizaje geográfico en un hábito divertido y visualmente atractivo.

---

## 🎯 Propósito y Visión

* **Promover la Cultura:** Acercar la diversidad del mundo a través de pistas que van más allá de lo obvio.
* **Reto Educativo:** Fomentar el aprendizaje diario con un formato de juego rápido y adictivo inspirado en Wordle.
* **Curiosidades del Mundo:** Compartir esos "datos curiosos" que hacen que cada país sea especial.

---

## 🎮 El Juego
Cada 24 horas, un nuevo país. La dificultad es progresiva:
1.  **Experto:** Información técnica o geografía física avanzada.
2.  **Difícil:** Datos de cultura general, historia y demografía.
3.  **Medio:** Curiosidades populares y economía.
4.  **Fácil:** Datos icónicos, banderas o monumentos reconocibles.

---

## 🛠️ Stack Tecnológico (Vibe Stack)

Gracias a la capacidad de interpretación de la IA, el proyecto cuenta con una arquitectura de última generación:

* **Framework:** [TanStack Start](https://tanstack.com/start) & React 19.
* **Estilos:** Tailwind CSS v4.
* **Componentes:** Radix UI para una accesibilidad total.
* **Build Tool:** Vite 7.
* **IA Stack:** Claude (Prompts & Lógica) + Lovable (Full-stack Development).

---

## ✨ Retos de Vibe Coding Resueltos

* **Validación Inteligente:** Motor de normalización que permite acertar ignorando acentos, mayúsculas o espacios extra.
* **Sincronización Temporal:** Lógica determinística para que el país cambie globalmente para todos los usuarios a la vez.
* **Sistema Bilingüe:** Interfaz y base de datos de pistas conmutables entre ES/EN en tiempo real.
* **Persistencia Local:** Seguimiento de intentos y victorias diarias mediante `localStorage`.
